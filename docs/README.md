## 🔧 技术架构概览

```
CBT Helper 架构
├── 前端 (Vue.js 3)
│   ├── 用户界面组件
│   ├── 路由管理
│   └── 状态管理
├── 服务层
│   ├── LLM集成服务
│   ├── 数据管理服务
│   └── API代理服务
└── 数据层
    ├── 本地存储 (LocalStorage)
    ├── 数据导入导出
    └── 状态持久化
```

## 🎨 设计系统

### 色彩主题
- **主色调**: 温暖绿色系 (#84A98C, #52796F)
- **强调色**: 温暖橙色 (#FF9B85)
- **背景色**: 柔和渐变 (天空→大地)

### 设计原则
- **温暖友好** - 圆角设计，柔和色彩
- **清晰直观** - 简洁的信息层次
- **响应式** - 适配各种设备尺寸
- **无障碍** - 考虑可访问性需求

## 🛡️ 安全与隐私

### 数据保护
- 本地优先存储策略
- 不向服务器发送敏感信息
- 用户完全控制数据
- 支持数据导出和删除

### API安全
- 客户端配置API密钥
- 请求代理和错误处理
- 限制敏感信息传输
